## Deskripsi Singkat
**Smart Attendance Quiz CLI** adalah aplikasi berbasis Python yang digunakan untuk mencatat kehadiran siswa dan memberikan kuis interaktif setelah absen.  
Seluruh proses berjalan di terminal tanpa menggunakan library eksternal.

---

## Fitur Utama
- Absensi otomatis
- Kuis pilihan ganda
- Penyimpanan data dalam tabel ASCII
- Mode siswa (Tanpa Nilai)
- Mode guru (Berpassword Guru@12345 yang bisa diganti)
- Multi mapel kuis
